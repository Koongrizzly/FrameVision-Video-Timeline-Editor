Core workflow

Built around a normal editing loop: import media, place clips, trim, preview, add effects, then render the real MP4 result.

📁

Project handling

Save and load project JSON files that preserve the edit structure.

  • Clip placement and timing
  • Tracks, transitions, fades, transforms and text clips
  • Multi-selection state and editor settings
💾

Autosave + recovery

Automatic recovery protects active work while keeping normal Save Project as the main habit.

  • Runs about every 60 seconds when changes exist
  • Recovery file for unsaved projects
  • Restore, delete, or ignore recovery on startup
🖼️

Project canvas

Set the final composition shape for preview and export.

  • Landscape, portrait and square presets
  • Canvas stays separate from preview quality
  • Source media and transition masks do not redefine the project canvas

Media Bin

A project media browser with enough polish to handle larger edits without losing track of files.

➕

Import and organize

  • Add videos, images, audio files and reusable Text assets
  • Duration detection through ffprobe when available
  • Editor-only rename keeps source files untouched
  • Open source folder, media info and copy media info
videoaudioimagetext
⭐

Favorites, sorting and dots

  • Favorites stay grouped above normal items
  • Sortable columns for name, type, duration, path and favorite state
  • Colored media-type dots for fast scanning
  • Thumbnail previews for images, text and cached timeline video thumbnails
🖱️

Drag/drop to timeline

  • Drag media directly onto any unlocked timeline track
  • Drop position maps to the shared timeline time model
  • Snap can adjust dropped clip timing
  • Dropping overlapping visual media refreshes transition/overlap overlays immediately

Timeline editing

The timeline supports real clip editing, track controls, grouped operations and practical navigation tools.

✂️

Clip operations

  • Move clips along time or between tracks
  • Trim left/right edges with source-duration safety
  • Split at playhead: keep both, left only or right only
  • Delete, duplicate, copy and paste
  • Reset trim to full source length
🧲

Navigation and snapping

  • Zoom in/out and zoom around playhead
  • Fit Timeline for a full-project view
  • Snap to useful timeline positions
  • Start, Play, Pause, Stop and End controls
  • Timeline slider and playhead scrubbing
🧱

Tracks and grouping

  • Add, remove, rename and reorder tracks
  • Hide/show visual tracks
  • Mute, solo, lock and set track volume
  • Multi-select clips for grouped move, delete, copy, paste and duplicate

Preview pane

The preview is both a viewer and a hands-on visual editing surface.

🔍

Inspect and navigate

  • Preview selected media, selected clips or full timeline playback
  • Pan and zoom inside the viewer up to about 50x
  • Viewer Fit and fullscreen controls
  • Source preview popup keeps the main timeline preview from being hijacked
🛡️

Heavy-preview safeguard

Demanding playback paths can show a small visual preparation notification instead of looking frozen.

  • Useful for backwards video preview and other heavy operations
  • Small animated loading feel without stealing timeline space
  • Designed as UI reassurance while preview work catches up
Preview is not the final render.

Heavy transitions, reverse playback and stacked effects can stutter in live preview while the exported MP4 remains normal.

Visual tools

Video and image clips can be transformed directly in the preview pane or through right-click menus.

🎛️

Transform controls

  • Move, resize, scale and rotate visual clips
  • Set opacity, fit, fill or stretch
  • Reset Transform returns a clip to default placement
  • X/Y readout shows offset from centered default position
🪞

Mirror / Flip

  • Mirror horizontal
  • Mirror vertical
  • Mirror both directions
  • Reset mirror state
  • Works on video and image clips
📍

Visual keyframes

  • Animate position, scale, rotation, opacity and fit mode
  • Add/update keyframe at playhead
  • Add Default Keyframe as a clean visual anchor
  • Delete nearest keyframe or clear all keyframes

Transitions

A PNG mask transition system for overlapping visual clips, including custom masks and direct transition replacement.

🧩

Overlap transitions

  • Transitions apply to overlapping visual clips
  • Transition blocks appear on the overlap area
  • Drop-added overlapping clips refresh transition overlays immediately
  • Duration can be set or clamped to the overlap
🖼️

Custom PNG masks

  • Transition files live in /assets/transitions
  • Refresh transitions from the panel or menu
  • Recommended: optimized 720p grayscale PNG masks
  • Light files improve live-preview smoothness
🔁

Transition controls

  • Drag a transition mask onto an existing transition block to replace it
  • Enable/disable a transition
  • Reverse transition direction
  • Invert mask, view info, test preview or remove transition

Text and titles

Reusable text assets plus clip-specific animation make it useful for titles, captions, lower thirds and credits.

🔤

Reusable Text assets

  • Create titles with font size, color, opacity, duration and position
  • Optional outline, shadow, background box and fades
  • Text assets appear in the Media Bin
  • Edit Text from the timeline or Edit Text Asset from the bin
✨

Clip-specific animation

  • Slide in/out from left, right, top or bottom
  • Zoom in/out
  • Typewriter / reveal characters
  • Rotate clockwise/counterclockwise
  • Pop, bounce and easing curves
  • Scroll up, down, left or right

Playback and audio

Clip playback, embedded video audio and separate audio tracks can be controlled without changing the original media files.

⏪

Reverse playback

  • Right-click video clip → Playback → Play backwards
  • Non-destructive: source file is not changed
  • Keeps normal timeline placement, trim and project structure
  • Embedded sound is included when usable and enabled
🔊

Audio controls

  • Mix all audio, video only, audio tracks only or mute
  • Clip volume, track volume and master volume
  • Track mute/solo and master mute
  • Detach audio from video clips
〽️

Fades and automation

  • Audio fade in/out with short, long or custom durations
  • Separate visual video fade in/out
  • Volume automation points and presets
  • Reset fades or clear automation

Export, performance and layout

Designed for practical editing: lower-quality live preview while keeping the final render clean.

📤

MP4 export

  • Final MP4 rendered from the full timeline through FFmpeg
  • Visual transforms, transitions, fades, text and audio mix are part of export
  • Export is the reliable result when preview struggles
⚡

Preview performance tools

  • Preview Quality: 480p, 720p, Full canvas or Auto
  • Cached thumbnail and waveform visual aids
  • Transition preview uses cached/lightweight paths where possible
  • Reverse-preview proxy preparation avoids bad forward-playback fallback
🧭

Layout presets

  • Editing: timeline-focused workspace
  • Timeline Top: timeline priority with panels below
  • Media Top: media, transitions and preview above timeline
  • Classic: media/transitions left, timeline and preview right
  • Reset Layout and adjustable editor font size
